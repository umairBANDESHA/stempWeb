# STEMP21 — Flutter Application

## Overview

**STEMP21** is a role-based school management application built using **Flutter** with **Firebase backend services**.
The system supports multiple user roles and provides controlled access to features based on authorization levels.

The project is designed with a **single codebase** targeting:

* Android (primary platform)
* Web (administrative access & testing)

---

## Core Features

### Role-Based Access Control

The application dynamically changes functionality based on user roles:

* **Admin**
* **Regional Director**

Each role has isolated permissions and dashboard visibility.


## Tech Stack

### Frontend

* Flutter (Dart)
* Material UI
* Responsive layouts (Mobile + Web)

### Backend

* Firebase Authentication
* Cloud Firestore

### Deployment

* Android APK / Play Store
